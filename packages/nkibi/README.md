# nkibi

A lightweight, simple terminal text editor for Nitpick. It loads a file entirely into memory and allows basic typing and editing. Inspired by `kibi`.

## Installation

```bash
make
```

## Usage

```bash
./nkibi <filename>
```

## Features
- Move cursor using Arrow keys.
- Enter/Backspace support.
- Live rendering with TUI panel.
- Save file instantly using `Ctrl-S`.
- Quit with `Ctrl-Q`.
